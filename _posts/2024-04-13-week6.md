# 증감 연산자



```cpp
#include <stdio.h>

void main()
  int a=20;

  a=a+10;
  printf("a= %d\n", a);
  a=a*10;
  printf("a= %d\n", a)'
  a=a-10;
  printf("a= %d\n", a);
  a=a/10;
  printf("a= %d\n", a)'

```

### 변수 a의 값에 10을 더한 후 결과를 변수 a에 저장합니다.

```cpp
#include <iostream>

/* run this program using the console pauser or add your own getch, system("pause") or input loop */

int main(int argc, char** argv) {
	int a=10;
	
	a++;
	a++;
	printf("a=%d\n", a);
	++a;
	++a;
	printf("a=%d\n", a);
	return 0;
}
```
